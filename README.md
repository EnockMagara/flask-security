
## How to run

### Create virtual environment and install dependencies

```
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Run the back end

```
python app.py
```

### Run the front end

In a **different terminal**, navigate to the project's directory and run

```
source .venv/bin/activate
python frontend.py
```
